# 2026-06-04 开发日志

## 今日目标

- 修复右侧图层面板中元素较多时无法滚动查看的问题。

## 已完成

- 在右侧图层面板中新增独立的图层列表滚动容器 `.layer-list`。
- 保持图层标题区域固定在面板顶部，前景/中景/背景分组内容在列表区域内滚动。
- 增加窄滚动条样式，保证暗色编辑器界面下仍能清楚查看和滚动大量图层项。
- 保留原有图层点击选中、缩略图、透明度显示、复制/删除和层级管理功能。
- 将右侧图层行中的三点图标替换为锁图标。
- 为拼贴元素新增 `locked` 状态，支持在图层面板和顶部工具栏切换锁定。
- 锁定后的元素不可拖拽、缩放、旋转、修改属性、移动图层顺序或被删除。
- 底部图层顺序按钮会在选中锁定元素时自动禁用。
- 新增撤销/重做历史栈，记录最近 80 步画布编辑状态。
- 支持 `Ctrl+Z` 撤销，`Ctrl+Y` 和 `Ctrl+Shift+Z` 重做。
- 移动、缩放、旋转、复制、删除、锁定、图层顺序、背景切换都会进入历史记录。
- 在顶部工具栏恢复撤销/重做 UI 按钮，按钮状态会根据历史栈自动启用或禁用。
