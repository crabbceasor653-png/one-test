## 已完成

- 完成阶段 3：元素编辑。
- `CanvasWorkspace` 中的 Konva 图片元素支持拖动。
- Konva Transformer 支持四角缩放和旋转。
- 拖动结束后将 `x`、`y` 写回 React 状态。
- 缩放 / 旋转结束后将 `width`、`height`、`rotation` 写回 React 状态。
- `RightPanel` 从静态示例数据改为读取真实画布元素。
- 右侧图层面板可显示真实画布元素，并支持点击选中。
- 右侧属性面板可显示和修改选中元素的 X、Y、宽度、高度、旋转角度和不透明度。
- 完成阶段 4：图层管理基础能力。
- 在顶层状态中新增图层顺序操作：置顶、上移、下移、置底。
- 支持选中元素切换到前景层、中景层、背景层。
- 画布渲染顺序改为背景层 -> 中景层 -> 前景层，同层内按 `zIndex` 从低到高渲染。
- 右侧属性面板新增图层下拉选择和同层顺序快捷按钮。
- 画布底部图层快捷按钮接入真实操作，未选中元素时禁用。
- 完成阶段 5：复制、删除、多选基础。
- 将选中状态从单个元素 ID 升级为多个元素 ID。
- 支持 Shift 点击画布元素进行多选 / 取消多选。
- 支持 Shift 点击右侧图层行进行多选 / 取消多选。
- 画布和图层列表会同时高亮多个选中元素。
- 浮动工具栏和右侧属性面板支持复制选中元素。
- 浮动工具栏和右侧属性面板支持删除选中元素。
- 支持 Delete / Backspace 删除选中元素。
- 支持 Ctrl+D / Cmd+D 复制选中元素。
- 完成阶段 6：导出 PNG 与背景模式。
- 新增 `CanvasBackground` 和 `CanvasBackgroundMode` 类型。
- 新增 `src/utils/export.ts`，使用离屏 Konva Stage 导出干净 PNG。
- 导出时按背景层 -> 中景层 -> 前景层顺序绘制，同层内按 `zIndex` 绘制。
- 支持透明、白色、自定义颜色三种背景模式。
- 顶部“导出 PNG”按钮接入真实下载逻辑，并在导出中禁用。
- 底部状态栏新增背景模式切换和自定义颜色选择器。
- 画布预览会同步背景模式；透明模式使用棋盘格提示透明区域。
- 完成阶段 7：整理与首版验收。
- 扫描并修复用户可见文本中的历史乱码残留。
- 上传 PNG 读取失败时显示基础提示。
- 空画布导出时显示“请先上传 PNG 并添加到画布”的提示，避免导出空作品。
- 导出失败时显示基础错误提示。
- 导出成功后显示下载提示。
- 修正 Delete / Backspace 快捷键在输入框、下拉框、颜色选择器中误删画布元素的问题。
- 收紧底部状态栏列宽和间距，降低 1280px 桌面宽度下的横向溢出风险。
- 新增全局轻量提示条 `app-toast`。

- 将“置于顶层 / 上移一层 / 下移一层 / 置于底层”从画布浮层移动到底部状态栏原缩放区域。

- 移除画布中的 `floating-actions` UI，让画板区域保持干净。
- 顶部“已放置 / 多选”操作条继续复用复制、删除等原有回调，不删除对应功能。

- 移除旧的 CSS 强制缩放 Konva canvas 规则，避免内部坐标和屏幕预览不同步。
