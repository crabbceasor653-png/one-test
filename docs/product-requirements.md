# 植被拼贴编辑器产品需求

## 产品目标

开发一款纯前端网页版图片拼贴工具。用户可以上传或使用 PNG 树木素材，通过拖拽、缩放、旋转、复制、分层排列，把多张树木 PNG 拼成具有前景、中景、背景层次的植被景观图。

## 第一版范围

第一版只做稳定可用的核心编辑能力，不一次性堆叠过多复杂功能。

必须完成：

- PNG 上传，并保留透明背景。
- 左侧素材栏展示上传素材缩略图。
- 素材可添加到画布。
- 画布元素支持拖动、缩放、旋转。
- 元素支持复制、删除、透明度调整。
- 元素支持背景层、中景层、前景层切换。
- 右侧属性面板可编辑选中元素属性。
- 右侧图层面板展示三类图层及元素列表。
- 支持导出最终画面为 PNG。
- 支持画布背景切换为透明、白色、自定义颜色。

第一版暂缓：

- 完整撤销 / 重做历史。
- 复杂组合编辑。
- 云端素材库。
- 自动保存到后端。
- 协作编辑。
- AI 生成素材。

## 用户角色

目标用户是非专业设计用户，需要界面直接、操作反馈清楚、控件名称容易理解。

## 关键交互

- 用户上传 PNG 后，缩略图立即出现在左侧素材栏。
- 用户点击或拖拽素材，将素材放入中央画布。
- 用户点击画布元素后，出现选中框和缩放 / 旋转控制点。
- 用户在右侧属性面板调整大小、角度、透明度、图层。
- 用户在右侧图层面板切换元素所属层级或选中元素。
- 用户点击导出 PNG，下载最终画面。

## 验收标准

- 上传透明 PNG 后，透明区域在画布和导出结果中均保持透明。
- 任一元素可以被选中、拖动、缩放、旋转。
- 修改右侧属性时，画布元素实时更新。
- 图层调整后，元素遮挡关系正确变化。
- 导出的 PNG 与画布可见结果一致。
- 页面在普通桌面浏览器中无明显布局错位。
